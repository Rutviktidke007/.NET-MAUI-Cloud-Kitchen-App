# Cloud Kitchen App - .NET MAUI

This is a cloud kitchen app developed using .NET MAUI. The app integrates camera functionality, location services, and mobile features such as swipe gestures, loading indicators, and a flyout menu. It is designed to provide an interactive, efficient, and user-friendly experience for managing cloud kitchen operations.

## Features

- **Camera Integration**: Capture images of food orders, dishes, or inventory directly from the app.
- **Location Services**: Use location tracking to manage kitchen delivery zones and customer locations.
- **Swipe Gestures**: Swipe functionality for easy navigation between sections and quick actions.
- **Loading Indicator**: Display a loading spinner during app data fetch or processing.
- **Flyout Menu**: A convenient flyout navigation menu for accessing app sections.

## Prerequisites

Before running the project, make sure you have the following installed:

- .NET 6 or later
- Visual Studio 2022 or later (with .NET MAUI workload)
- Android/iOS emulator or physical device for testing
- Android/iOS SDK (for mobile development)
